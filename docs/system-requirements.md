# System requirements

System requirements

To ensure optimal viewing and functionality of this website's static
HTML pages, please ensure your system meets the following requirements.
⚠️ Note: While other modern browsers may render the pages correctly,
full compatibility is guaranteed only with Chrome and Firefox.

#### a. Supported Browsers:

Google Chrome (130.0.x.x and above/latest stable version) Mozilla Fire
Fox (130.x.x and above/latest stable version)

#### b. Operating Systems:

Windows 10 or later macOS 11 (Big Sur) or later Linux distributions with
a modern desktop environment (e.g., Ubuntu)

#### c. Hardware Requirements:

Processor: Dual-core CPU or higher Memory: Minimum 2 GB RAM Display:
Minimum resolution of 1024×768 pixels

#### d. Additional Requirements:

JavaScript Enabled: Required for interactive elements Cookies Enabled:
For session management and preferences Internet Connection: Required to
access hosted pages and resources
